TURTLE COMANDOS

makeTurtle(x, y)

Cria uma nova tartaruga na posição x/y

penUp()

Levanta a caneta

penDown()

Abaixa a caneta

setWeight(weight)

Muda a espessura da caneta

setColor(color)

Muda a cor da caneta. Aceita nomes de cores em inglês 'blue' ou código hexadecimal '#FF00FF'

forward(distance)

Move para frente determinados pixels

backward(distance)

Move para trás determinados pixels

goto(x, y)

Move para posição x/y

face(degrees)

Aponta para determinado grau: direita = 0, baixo = 90, esquerda = 180, cima = 270

right(degrees)

Gira para direita determinados graus

left(degrees)

Girar para esquerda determinados graus
